# 4 Weeks Strength and Conditioning Plan

### Week 1

```
Day 1: 3-5 rounds of dsw + dsn + dfsq + dc&p + pushup (x10)

Day 2: alternate dc&p and pullup (x5) for 15' emom

Day 3: 2x 2hsw + 2x 1hsw (R+L) + 2x thruster (R+L), ladder up +2 reps for 20'
```

### Week 2

```
Day 1: 5x dcln + 30'' rack hold + 5x dcln + 30'' rack hold, amrap 20'

Day 2: 5x dfsq + 15'' quarter squat hold + 15'' half squat hold + 15'' bottom squat hold + 5x dfsq, amrap 20'

Day 3: 1hsw + 1hcln (l+r) + 1hsn (l+r) + 1hjerk (l+r) + reverse lunge (l+r) (x5), amrap 20'
```

### Week 3

```
Day 1: 8x dcln + 5x dfsq + 5x dcln + 3x dfsq + 3x dcln + 2x dfsq + 2dcln + 1x dfsq, amrap 20'

Day 2: pressing get up (press at every stage of the get up) for 15'

Day 3: 10x dfsq + 10x press, amrap 20'
```

### Week 4

```
Day 1: 5x press + 30'' overhead hold + 5x dfsq + 30'' rack hold, amrap 20'

Day 2: 5x dkb lounge + 5x press (l+r), amrap 20'

Day 3: dsw + dsn + dc&p + dfsq + pushup (x5), amrap 20'
```
