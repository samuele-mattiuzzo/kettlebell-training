# Cluster training

- Select a weight that is 80-90% of 1RM
- 1 cluster > 5 singles with 10-15 sec pause in between + 3' rest after
- Complete 5 clusters (5 x 5)
- Break in: use 3RM, do 5 x 3 with 1' breaks, till 5x5, then work breaks down to 45'' down to 15
- Continuation: 5 singles > 5 doubles > 5 triples (5x)

- D1: Squat 5x5, snatch 5x5, get ups 2x5 (regular) / D2: Press 5x5, bent over row 5x5, windmill 2x5 (regular)

# Russian Bear

- heavy bells 3-6 reps max
- set 1: all out 6 reps (3' rest) OR 1-3RM do 5 x cluster reps
- set 2: 4-5 reps (3' rest) OR drop from 1-3RM and to 6 reps regular style
- set 3-20: 3 reps (1' rest) with same weight as above
- A: Press, Bent Over Rows, Squats / B: Double Snatch, Press, Renegade Row / C: Press, Bent over Row, Lunge

# HIT

- 3 x 5 (2 low rep sets, last one to failure)
- works for about 4 weeks
- when amrap > 7+ reps, heavier bells or harder exercise
- Press + Row + Squat + Swing / Floor press + renegade row + Rev Lunge + Snatch / See Saw Press + Row + Squat + Swing
